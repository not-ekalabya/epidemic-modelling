### Getting Started

```pip install -r requirements.txt```

```python preprocessing.py```

```python model.py```

Modify ```COUNTRY_ISO2 = "BD" ``` and ```COUNTRY_ISO3 = "BGD" ``` with ISO-2 and ISO-3 country codes at ```main.py``` to access population densities of different countries.
